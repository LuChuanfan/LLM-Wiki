# Claude Code 6个必装 Skills 推荐

来源：微信公众号文章（卡兹克、可达），推荐了 6 个实用的 Claude Code Skills。

## 推荐列表

### 1. Frontend Design（前端设计）
Anthropic 官方出品，排名第一。解决 AI 做前端的"AI 味"问题——禁止烂大街字体和紫色渐变，要求先定美学方向再写代码。能力越差的模型提升越明显。

### 2. 办公四件套（docx/xlsx/pdf/pptx）
Anthropic 官方出品。不装时 Agent 每次从零摸索排版，装上后自带文档处理流程和代码模板，输出质量大幅提升。可叠加 Frontend Design 提升 PPT 颜值。

### 3. Web Access Skill
@一泽 开发。通过 CDP 直连本地 Chrome，带登录态，可访问小红书/B站/微博等站内内容。支持 Jina 预处理节省 token，自动沉淀每个网站的操作经验。

### 4. PUA
专治 AI 摆烂。四级压力升级机制，卡住时强制换思路。v3 版本整合了十几家公司的方法论。建议卡住时手动 `/pua`，不开默认模式。

### 5. Claude-mem
解决记忆持久化。自动记录对话关键信息，新会话自动注入上下文。三层渐进式检索省 Token。本地 Web 界面可视化查看记忆。

### 6. Skill-Creator（最重要）
帮你把自己的需求和经验封装成 Skills。作者核心观点：**最牛的 skill 永远是你自己造的那个。** 通用技能在贬值，"知道自己需要什么"的能力在升值。

## 关键洞察

> "Skills 不是装的越多越好，越精简越科学，效果才越好。"

> "真正贬值的是通用技能。真正升值的，是你知道自己需要什么的那个能力。"
